# 日経225先物 短期売買サポートWebアプリ

過去約30年の市場データから、**「いまと似た状況のとき、翌日・当日に上がった確率は何%だったか」** を提示するWebアプリケーション。

> ⚠️ **免責事項**
> 本アプリは過去データの統計的な集計結果を表示する参考ツールであり、将来の値動きを保証するものではありません。
> 投資助言・売買推奨ではなく、最終的な売買判断と責任は利用者に帰属します。

## 特徴

- **統計エンジン**: ルールベースの条件から過去30年のデータを検索し、マッチした事例の騰落確率を表示
- **透明性重視**: ブラックボックスのAI予測ではなく、「何を根拠にその数字が出たか」を全て開示
- **条件ビルダー**: 「N日連続下落」「ギャップダウン」など、カスタム条件で統計検索可能
- **事例ビューア**: マッチした過去事例をチャート付きで確認
- **フィルタリング**: イベント日（FOMC、日銀会合、SQ等）や季節性で絞り込み

## ダッシュボード機能

3つの時間軸で買い優位性スコアを表示：

```
今買う？     [やや優位 62%]  n=142  
今日買う？   [中立   51%]  n=310  
明日買う？   [優位   68%]  n=89
```

各スコアに付随する情報：
- サンプル数（n）
- 上昇時平均リターン・下落時平均リターン
- 最大ドローダウン
- 95%信頼区間

## 統計の誠実性

以下をコンポーネントレベルで強制します：

1. n < 30 の確率には常に「サンプル不足」警告
2. 確率に95%信頼区間（Wilson区間）を併記
3. 期間分割の再現性チェック自動表示
4. 手数料・スリッページを控除した実質期待値も表示
