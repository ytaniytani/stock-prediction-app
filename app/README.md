# 日経225先物 予想ラボ

短期売買の参考にするための統計ツール。過去データの中から現在の相場状況と似た日を探し、その後の値動きの傾向を確率・信頼区間つきで提示する。詳細仕様は `../SPEC.md` を参照。

## セットアップ

```bash
npm install
npm run dev       # 開発サーバー
npm run build     # 型チェック＋本番ビルド
```

## データについて

このアプリはブラウザ内（IndexedDB）で完結するフロントエンドのみの構成。初回起動時は**架空のサンプルデータ**が自動生成される（統計的な性質だけを模した乱数データで、実際の日経平均・ダウ等の値ではない）。

信頼できる分析には、「データ管理」画面から実際の日経平均・日経225先物・NYダウ・NASDAQ総合・TOPIX・ドル円のCSV（Yahoo!ファイナンス形式・investing.com形式に対応）を取り込むこと。取り込むとデータモードが自動的に「実データ」に切り替わる。

## 主な機能

- **ダッシュボード**: 「今」「今日」「明日」それぞれの上昇確率をk近傍法で判定
- **パターン検索**: 条件ビルダー＋イベント/季節性チェックボックスで過去30年から該当日を検索し、上昇確率・信頼区間・分布・期間別安定性を表示
- **事例ビューア**: 該当日ごとのチャートと予想→答え合わせ
- **時代別分析**: 金融環境（レジーム）別のパターン成績比較
- **バックテスト**: 仮想資産曲線・勝率・最大ドローダウン・ケリー基準・損切り前提の期待値
- **イベント暦**: 今後の日銀会合/FOMC/雇用統計/CPI/SQ/選挙日程と、過去の平均騰落率
- **タイムマシン**: 「今日」を過去の任意の日に設定して先読みなしで判定を再現し、実際の結果と答え合わせ。ウォークフォワード検証で判定手法そのものの過去の的中率を検証できる
- **紙トレ記録**: 仮想売買の記録と成績振り返り
- **データ管理**: CSVインポート、サンプル/実データ切り替え、取引コスト設定

## 免責

過去データの統計的な集計結果を表示するツールであり、将来の値動きを保証しない。投資助言・売買推奨ではない。
