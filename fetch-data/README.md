# 実データ取得スクリプト（ID・パスワード不要）

「日経225先物 予想ラボ」用に、日経平均・NYダウ・NASDAQ総合・TOPIX・ドル円の日足データを、
**ログイン不要の公開CSV**（[stooq.com](https://stooq.com)）から取得するNode.jsスクリプト。

⚠️ Yahoo!ファイナンス等のID・パスワードを入力させる方式は、パスワード漏えいのリスクと利用規約上の懸念があるため採用していません。stooqは無料・匿名で参照できる公開データを使っています。個人利用の範囲で、1日1回程度の実行を目安にしてください。

## 必要環境

- Node.js 18以上（グローバル`fetch`を使用。アプリ本体（`../app`）と同じバージョンで動作確認済み）

## 使い方（PC）

```bash
cd stock-prediction/fetch-data
node fetch.mjs
```

`./out/N225.csv` `./out/DJI.csv` `./out/IXIC.csv` `./out/TOPX.csv` `./out/USDJPY.csv` が生成されるので、アプリの「データ管理」画面のCSVインポート欄にドラッグ&ドロップ（またはファイル選択）してください。

## 使い方（アプリと連携・推奨）

サーバーモードで起動しておくと、アプリの「データ管理」画面のボタン一つで全銘柄を取り込めます。

```bash
node fetch.mjs --serve
```

起動すると `http://localhost:8787` でCSVを配信します。アプリの「データ管理」→「ローカル取得サーバーから読み込む」でサーバーURLを確認し、「サーバーで最新化してから取り込む」を押すだけです。

このプロセスは起動したままにしておくと、24時間ごとに自動で再取得もします。終了するには `Ctrl+C`。

## iPhoneなど別端末から使う場合

`--serve` 起動時にターミナルへLAN内IPアドレス（例: `http://192.168.1.23:8787`）が表示されます。

1. PCで `node fetch.mjs --serve` を起動したままにする
2. iPhoneをPCと**同じWi-Fi**に接続する
3. iPhoneでアプリを開き、「データ管理」画面のサーバーURLを上記のLAN内IPアドレスに変更する

外出先など別ネットワークからはこの方法では届きません。将来的にどこからでも使いたくなった場合は、このサーバーをインターネット上のどこか（自宅サーバー、クラウドの無料枠など）に常時起動しておく形に発展させる必要があります。必要になったら相談してください。

## 銘柄シンボルについて

`fetch.mjs` 冒頭の `TICKERS` オブジェクトでstooqのシンボルを指定しています。取得に失敗する場合は [stooq.com](https://stooq.com) で銘柄を検索し、URLの `s=` の値に書き換えてください。

## 日経225先物（N225F）について

無料・ログイン不要で日経225先物の過去データを配信している場所が見当たらなかったため、このスクリプトの対象外です。アプリ側は先物データが無い場合、自動的に日経平均株価（N225）で代用して統計計算を行います。より正確に分析したい場合は、証券会社のツール等からエクスポートしたCSVを「データ管理」画面から個別に取り込んでください。
